Executive summary

LifeQuest delivers a full end-to-end learning experience: structured financial literacy curriculum, interactive assessment, business simulation game, city-building progression, skill tracking, and server-side scoring. All six quest levels (48 stages) are implemented with content, progression rules, backend functions, and a production web deployment.

Content delivery

  • Lessons 1–6: Fully hand-authored content (read passages, quizzes, game settings) per age band.
  • Lessons 7–48: Stage-specific content library — each of 42 template lessons has unique teaching copy and quiz questions aligned to its title and learning objective.
  • Teen track (ages 13–17, Levels 4–6): 30 bespoke variants with real-world scenarios, AED-based exercises, and advanced quiz questions.
  • Lesson flow: Read → Quiz (5 questions) → Lemon City business game → Reward & progression.
  • Currency & locale: AED throughout; UAE-appropriate examples and context.

Core product features (completed)

Learning & assessment

  • Four-phase lesson session (Read, Quiz, Game, Reward)
  • Five quiz questions per lesson with cognitive progression
  • Hearts-based quiz UX with explanations on each answer
  • Quiz success feedback (score animation, sound effects)
  • Server-validated quiz scoring and answer keys
  • Adaptive AI lesson personalisation (lessons 7+, signed-in users)
  • Offline template fallback when AI unavailable

Progression & skills

  • Linear stage unlock — learners never get stuck between stages
  • Level unlock requires completion + 80% average quiz mastery
  • Onboarding screening → placement on levels 1–6
  • Concept skills tracking (L1–L6) updated after each lesson
  • Progress dashboard with skill bars and Business IQ metrics
  • Time gates defined for upper levels (14 / 30 / 90 / 180 days)

Lemon City business game

  • Interactive lemonade stand simulation in every lesson
  • Teaches revenue, cost, pricing, and profit
  • Per-lesson difficulty tuned to curriculum stage
  • Game difficulty scales with learner skill mastery
  • Results submitted to server for scoring and analytics

City gamification

  • 48 building plots across 6 themed districts
  • Plots unlock as lessons are completed
  • Tower rewards earned on lesson completion
  • Visual progress map tied to curriculum levels
  • Brand-aligned UI with mascot guides (Penny, FinBot, Atlas)

Scoring & rewards economy

  • XP, coins, and LQ score — earned on lesson completion
  • Business IQ — three dimensions (profit, decision, resilience) updated from quiz and game performance
  • Streak tracking — daily engagement rewards
  • Badges — milestone achievements (e.g. first profit)
  • City towers — visual rewards persisted to user profile
  • Idempotent server functions — safe retries, no duplicate rewards

Backend & infrastructure (completed)

  • Firebase Authentication — child, parent, teacher role support
  • Cloud Firestore — profiles, progress, stats, badges, city state
  • Cloud Functions (europe-west1):
    • runScreening — placement assessment and skill seeding
    • completeLesson — full scoring, skills update, rewards
    • generateAdaptiveLesson — AI-personalised content with 7-day cache
    • submitGamePlay — game result recording
  • Shared types package — consistent schemas between client and server
  • Firebase Hosting — production web app deployed
  • Deploy scripts — automated build and release pipeline
  • 32 automated tests — curriculum, progression, content, quiz, city coverage

User experience & polish (completed)

  • Brand green theme aligned across app
  • Onboarding flow (age, guide selection, screening)
  • Learn screen with level gates and mastery hints
  • Progress screen with quest skills and Business IQ
  • Sound effects on web and native (quiz bells, success chimes)
  • Responsive web layout for classroom and home use
  • Client-side fallbacks when cloud functions temporarily unavailable

Adaptive learning pipeline (completed)

  • AI generates personalised read/quiz content for lessons 7+ (handcrafted 1–6 protected)
  • Learner mastery level (concept skills) adjusts AI difficulty tier
  • Curriculum-aligned seed content ensures learning objectives stay on track
  • Server-side answer key caching for quiz validation
  • Content safety filter on all AI output
  • Deterministic template fallback — app works fully without AI API keys
