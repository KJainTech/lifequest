# City art — Kenney CC0 (recommended)

LifeQuest uses a **Kenney-inspired** isometric renderer in code. For production sprite sheets,
add PNGs from Kenney's free packs (CC0, no attribution required):

- [Isometric Tiles City](https://kenney.nl/assets/isometric-tiles-city)
- [Isometric City](https://kenney.nl/assets/isometric-city) (128× buildings)

Drop extracted PNGs here and wire via `CitySpriteAtlas` when ready.
